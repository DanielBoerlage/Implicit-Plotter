#Implicit Plotter
Implicit/General Equation Plotter by Daniel Boerlage and Philip Xu  2014

Supported Types:
	* Explicit function

Future supported types:
	* General equation
	* Parametric functions
	* Polar functions/equations

**In hard development; no official binaries released yet**
